#### Town Hall Summary:
The community is discussing proposals for round two of the program.
Meeting key points:

-The program is an experiment in governance, reputation management, and incentivization.
-Deep Funding is a treasury that funds proposals to improve the Singularity platform.
-In round one, community members submitted proposals for how to improve Deep Funding.
-In round two, the community will vote on a shortlist of proposals to determine which ones will be implemented.

One proposal is to create a pool of rewards for onboarding existing AI services to the Singularity platform.
There are questions about how the rewards for onboarding existing services would be determined. Some ideas include:
A fixed amount for each service onboarded.
A separate pool for API readiness work.
Only rewarding developers for entirely new services, not existing ones.
